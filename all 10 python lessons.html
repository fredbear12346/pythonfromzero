Getting Started

Install Python, write your first program, and understand how print() works.

Step 1: Install Python

  1. Go to python.org
  2. Click "Downloads" ‚Üí Download the latest version
  3. Run the installer
  4. IMPORTANT: Check "Add Python to PATH" before clicking Install!

Step 2: Your First Program

Open any text editor (Notepad, VS Code, etc.) and type:

print("Hello World!")

Save the file as hello.py

Open your terminal/command prompt and run:

python hello.py

You should see: Hello World!

Understanding print()

print() displays text on your screen. Whatever you put inside the parentheses gets shown.

print("Python is fun!")
print(2 + 2)
print("My age is", 25)

üí° Try This:

Change the text inside print() to say something about yourself. Run it again!

Common Mistakes

  • Print("Hello") ‚ùå Capital P doesn't work. Use lowercase print
  • print("Hello" ‚ùå Missing closing parenthesis
  • print(Hello) ‚ùå Text needs quotes: "Hello"

Variables

Store data in named containers. Learn about types and the assignment operator.

What is a Variable?

A variable is like a labeled box. You put a value inside, and you can use the label to get it back later.

name = "Alice"
age = 25
height = 5.6
is_student = True

‚ö†Ô∏è Important:

The = sign does NOT mean "equals" like in math. It means assign. It puts the value on the right into the variable on the left.

Data Types

Type Example Description
str"Hello"Text (string)
int25Whole number
float5.6Decimal number
boolTrue / FalseTrue or False value

Using Variables

name = "Bob"
age = 30

print("Name:", name)
print("Age:", age)
print("Next year:", age + 1)

üí° Variable Naming Rules:

  • ‚úÖ Start with a letter or underscore: name, _age
  • ‚úÖ Use letters, numbers, underscores: user_name1
  • ‚ùå No spaces: my name is invalid
  • ‚ùå Can't start with number: 1name is invalid
  • ‚úÖ Use lowercase with underscores: first_name

Strings

Work with text: create, combine, format, and use string methods.

Creating Strings

Use single quotes ' or double quotes ":

name = "Alice"
message = 'Hello'
text = "It's a great day"
quote = 'She said "Hi"'

f-strings (String Formatting)

The easiest way to insert variables into strings:

name = "Alice"
age = 25

print(f"Hello, {name}!")
print(f"You are {age} years old")
print(f"Next year: {age + 1}")

String Methods

text = "  Hello World  "

print(text.lower())      # "  hello world  "
print(text.upper())      # "  HELLO WORLD  "
print(text.strip())      # "Hello World"
print(text.replace("World", "Python"))

name = "alice"
print(name.capitalize()) # "Alice"

sentence = "a,b,c"
print(sentence.split(","))  # ['a', 'b', 'c']

Indexing

word = "Python"

print(word[0])   # 'P' (first character)
print(word[1])   # 'y'
print(word[-1])  # 'n' (last character)
print(word[0:3]) # 'Pyt' (characters 0, 1, 2)
print(len(word)) # 6 (length)
Lesson 4 of 10

Numbers & Math

Integers, floats, operators, and the math module.

Basic Operators

add      = 5 + 2   # 7
subtract = 5 - 2   # 3
multiply = 5 * 2   # 10
divide   = 5 / 2   # 2.5 (always float!)

floor_div = 5 // 2  # 2 (whole number division)
modulo    = 5 % 2   # 1 (remainder)
power     = 5 ** 2  # 25 (5 squared)

The math Module

import math

print(math.sqrt(16))      # 4.0
print(math.pi)            # 3.14159...
print(math.ceil(4.2))     # 5 (round up)
print(math.floor(4.8))    # 4 (round down)
print(math.pow(2, 3))     # 8.0

Converting Between Types

# String to number:
num = int("42")        # 42
decimal = float("3.14") # 3.14

# Number to string:
text = str(42)         # "42"

# Float to int (truncates):
whole = int(3.99)      # 3 (not 4!)

üí° Common Mistake:

5 / 2 gives 2.5 (float), not 2. Use // for integer division: 5 // 2 = 2

Lists

Store multiple items, access by index, add, remove, and loop through lists.

Creating Lists

fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]
mixed = ["hello", 42, 3.14, True]
empty = []

‚ö†Ô∏è Remember:

Python starts counting at 0, not 1! This is the #1 beginner mistake.

Accessing & Modifying

fruits = ["apple", "banana", "orange"]

print(fruits[0])   # "apple"
print(fruits[-1])  # "orange" (last)
print(len(fruits)) # 3

# Change an item
fruits[0] = "mango"

# Add items
fruits.append("grape")
fruits.insert(1, "kiwi")

# Remove items
fruits.remove("banana")
deleted = fruits.pop(0)

Looping Through Lists

fruits = ["apple", "banana", "orange"]

# Method 1: Loop through items
for fruit in fruits:
    print(f"I like {fruit}")

# Method 2: Loop with index
for i in range(len(fruits)):
    print(f"{i}: {fruits[i]}")

# Method 3: Using enumerate
for index, fruit in enumerate(fruits):
    print(f"{index}: {fruit}")
Lesson 6 of 10

Dictionaries

Store key-value pairs. Access data by name, not by position.

What is a Dictionary?

person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

print(person["name"])  # "Alice"
print(person["age"])   # 25

Adding & Changing

person = {"name": "Alice", "age": 25}

# Change value
person["age"] = 26

# Add new key
person["city"] = "Boston"

# Remove key
del person["age"]
value = person.pop("city")

# Safe access
age = person.get("age", 0)  # Returns 0 if not found

Looping Through Dictionaries

person = {"name": "Alice", "age": 25, "city": "NYC"}

# Loop through keys
for key in person:
    print(key)

# Loop through values
for value in person.values():
    print(value)

# Loop through key-value pairs
for key, value in person.items():
    print(f"{key}: {value}")

If / Else

Make decisions in your code. Run different blocks based on conditions.

‚ö†Ô∏è Most Important Rule:

Python uses indentation (spaces at the start of a line) to know what code belongs inside an if statement. This is NOT optional!

if age >= 18:
    print("Adult")  # ‚Üê 4 spaces of indentation
print("Done")  # ‚Üê No indentation, outside if

If / Elif / Else

age = 20

if age < 13:
    print("Child")
elif age < 18:
    print("Teen")
elif age < 65:
    print("Adult")
else:
    print("Senior")

# Output: Adult

Comparison & Logical Operators

# Comparison: == != > < >= <=
5 == 5   # True
5 != 3   # True
5 > 3    # True

# Logical: and or not
age = 25
has_id = True

if age >= 18 and has_id:
    print("Can enter")

if age < 18 or age > 65:
    print("Discount eligible")

if not False:
    print("This runs")

Loops

Repeat code automatically with for loops, while loops, and range().

For Loop

fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print(f"I like {fruit}")

# Using range()
for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

for i in range(1, 6):
    print(i)  # 1, 2, 3, 4, 5

for i in range(0, 10, 2):
    print(i)  # 0, 2, 4, 6, 8

While Loop

count = 1
while count <= 5:
    print(count)
    count = count + 1

# Be careful of infinite loops!
# while True:
#     print("This never stops!")

Break and Continue

# BREAK: Exit loop early
for i in range(10):
    if i == 5:
        break
    print(i)  # 0, 1, 2, 3, 4

# CONTINUE: Skip to next iteration
for i in range(5):
    if i == 2:
        continue
    print(i)  # 0, 1, 3, 4 (skips 2)

Functions

Write reusable code with def, parameters, return values, and scope.

Defining Functions

def greet(name):
    print(f"Hello, {name}!")

greet("Alice")
greet("Bob")

def add(a, b):
    return a + b

result = add(5, 3)
print(result)  # 8

Default Parameters

def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"

print(greet("Alice"))           # Hello, Alice!
print(greet("Bob", "Hi"))       # Hi, Bob!

def create_user(username, active=True):
    return {"username": username, "active": active}

Variable Scope

global_var = "I'm global"

def my_function():
    local_var = "I'm local"
    print(local_var)      # Works
    print(global_var)     # Also works

my_function()
print(global_var)  # Works
# print(local_var) # ERROR! Doesn't exist here

File Handling

Read from and write to files. Work with text data and persist information.

Reading Files

# Recommended way (auto-closes)
with open("myfile.txt", "r") as file:
    content = file.read()
    print(content)

# Read line by line
with open("myfile.txt", "r") as file:
    for line in file:
        print(line.strip())

# Read all lines into list
with open("myfile.txt", "r") as file:
    lines = file.readlines()

Writing Files

# Write (overwrites existing!)
with open("output.txt", "w") as file:
    file.write("Hello, World!\n")
    file.write("Second line\n")

# Append (adds to end)
with open("output.txt", "a") as file:
    file.write("This is appended\n")

# ‚ö†Ô∏è "w" mode DELETES existing content!

File Modes

ModeDescription
"r"Read (default, file must exist)
"w"Write (creates or OVERWRITES)
"a"Append (adds to end)
"x"Create (fails if exists)
