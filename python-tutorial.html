The number one rule for beginners: Don't try to memorize everything. Just type things, see what happens, and break things on purpose. Making mistakes is how you learn.

1. Your very first program

This is the traditional first program everyone writes.

print("Hello World!")

Run this. Congratulations, you are now a Python programmer.

print() is a command that puts whatever is inside the parenthesis onto your screen.

print(2 + 2)
print("I am learning code!")

2. Variables

A variable is just a box to store a value. You give it a name, and put something into it.

name = "Alice"
age = 25
is_hungry = True

Important: The equals sign = does NOT mean "equals" like in math. It means assign. Put the thing on the right, into the name on the left.

You can use your variables later:

print("Your name is", name)
print("Next year you will be", age + 1)

You can change variables whenever you want:

age = 25
age = age + 1
print(age) # This will print 26

3. Basic Math

Python makes a very good calculator.

add       = 5 + 2
subtract  = 5 - 2
multiply  = 5 * 2
divide    = 5 / 2

# Two special useful ones:
remainder = 5 % 2   # gives 1, the remainder when you divide 5 by 2
power     = 5 ** 2  # 5 squared, equals 25

4. If Statements: Make decisions

This is how your code can do different things depending on the situation.

age = 17

if age >= 18:
    print("You are an adult")
else:
    print("You are a kid")

This is the single most important rule for beginners:

Notice the spaces at the start of the line, the indentation. This is not just to make code look pretty. Python cares about this. This is how Python knows what code belongs inside the if.

You can also check multiple things:

score = 85

if score >= 90:
    print("A grade")
elif score >= 80:
    print("B grade")
elif score >= 70:
    print("C grade")
else:
    print("Fail")

5. Lists

Store multiple things in order, in one place.

fruits = ["apple", "banana", "orange"]

To get an item out:

print(fruits[0]) # gets the FIRST item.

Important: Python starts counting at ZERO. This is the #1 cause of all beginner bugs. Always remember 0 is first, 1 is second, 2 is third.

Add new items to your list:

fruits.append("grape")
print(fruits)

6. Loops: Do something many times

The most useful loop is for loop. It runs once for every item in a list.

fruits = ["apple", "banana", "orange"]

for fruit in fruits:
    print("I like to eat", fruit)

If you just want to repeat something 5 times:

for i in range(5):
    print("Hello!")

7. Functions: Reuse your code

A function is a block of code you write once, then use many times. You make a function with def

def say_hello(name):
    print("Hello", name)
    print("Welcome!")

# Now I can use it as many times as I want!
say_hello("Bob")
say_hello("Alice")

Functions can also give a value back to you:

def add_two_numbers(a, b):
    result = a + b
    return result

sum = add_two_numbers(10, 15)
print(sum)

That is it. That is 80% of Python.

Seriously. Everything else you will ever learn is just a more advanced version of these concepts. All programming, no matter how complicated, is just combining these simple building blocks.

Right now the best thing you can do is spend an hour just playing with these. Change them. Break them. See what error message you get when you do something wrong. Errors are your friend, they are trying to help you.
